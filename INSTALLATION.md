# 📱 00o ইনস্টলেশন গাইড

## Windows এ APK তৈরি

1. Android Studio ডাউনলোড করুন
2. প্রজেক্ট খুলুন
3. Build → Build APK(s)
4. APK পান: app/build/outputs/apk/debug/app-debug.apk

## Android ফোনে ইনস্টল

1. APK কপি করুন ফোনে
2. Settings → Security → Unknown Sources চালু করুন
3. APK ট্যাপ করুন
4. Install করুন
